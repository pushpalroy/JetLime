# JetLime
A Kotlin Multiplatform library providing timeline UI components for Compose (Android, Desktop, JS, WASM, iOS).

## Features
- Vertical & horizontal timelines
- START / CENTER / END point placements
- Extended events with dual content slots
- Custom point types (filled, empty, icon)
- Multiplatform targets (Android, Desktop, JS, WASM, iOS)
